# PopTartee Configuration Files

## Files in this directory:

- **.env** - Environment variables (bot token, API keys, channel IDs)
- **system_prompt.txt** - Comprehensive system prompt with safety guidelines
- **discord_bot_token.secret** - Bot token backup (same as in .env)

## Security:

All files in this directory should be kept secure (chmod 600).
Never commit .env or .secret files to git.
